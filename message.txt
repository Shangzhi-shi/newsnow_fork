# 项目源地址
https://github.com/ourongxing/newsnow

# 启用 Corepack，这是 Node.js 官方的包管理器管理工具
# 它允许在不全局安装的情况下使用 pnpm、yarn 等包管理器
corepack enable

# 安装项目所有依赖包
# 相当于 npm install，但使用 pnpm 包管理器，拥有更好的性能和磁盘空间利用率
pnpm i

# 启动开发服务器
# 这会运行 package.json 中定义的 dev 脚本，通常会启动 Vite 开发服务器
# 启动后可以在本地浏览器访问应用（默认为 http://localhost:3000 或 http://localhost:5173）
pnpm dev
